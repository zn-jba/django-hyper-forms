Description

Now, when our visitors see the message, we can work on the member registration form.

Objectives

Create a form containing three fields. The first one is name. It should be of the text type and contain the your name label. The next one is age. Make it of the number type with the your age label. Since we try to set up a reading club, ask newcomers about their favorite book. Add the third field and call it favorite_book. Assign the your favorite book label to it. You can represent it in any way you'd like, but the form page should be reachable at the /register URL. Add these three fields to your form and add the Submit button.

You can use the Django built-in Form class to create a form instance and then put it to an HTML template of the page. At this point, we need to print the data to the terminal, so iterate it over the form data in the corresponding view and print it in the following format:

%field_name_1%: %value_1%
%field_name_2%: %value_2%

Examples

Example 1: An example of the output

name: Bob
age: 19
favorite_book: Romeo and Juliet
