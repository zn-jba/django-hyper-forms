Description

Questionnaires are useful when you need to collect certain information from your friends or clients. That's why it's always a good idea to add a form or two on your website.

Imagine, you are planning to set up a reading club. All prospective members need to fill out a simple form as a brief introduction, so you decided to create a website to collect this data.

Let's begin with creating a welcome page for your website.

Objectives

Make sure your server renders the page at the localhost:{port}/ address.

Let's put the welcome line inside the <h2> tag, the description inside the <p> tag, and emphasize the coming soon line with the <b> tag. Use the built-in render function to return the rendered template.

Then run the application using the python manage.py runserver command. Make sure the page is displayed correctly.
